权益详情页面说明文档

1. 页面概述

权益详情页面展示用户已购买的单个权益包的详细信息，包括权益内容、使用进度、使用说明和购买记录等，并提供使用和转赠功能。

2. 功能模块

2.1 顶部导航栏

  • 返回按钮：点击返回我的权益页面
  • 页面标题：显示"权益详情"
  • 更多按钮：提供额外操作选项

2.2 权益头部信息

  • 使用渐变背景、图标和大标题突出显示权益包名称
  • 显示权益包价格和有效期信息
  • 右侧标签展示权益状态（有效期内/即将到期/已过期）

2.3 包含权益列表

  • 分项展示权益包中包含的各项具体权益
  • 每项权益显示：图标、名称、说明和使用进度
  • 使用进度条直观展示权益使用情况
  • "查看记录"链接跳转至对应权益的使用记录页面

2.4 使用说明

  • 提供权益使用的具体步骤和注意事项
  • 帮助用户正确使用各项权益

2.5 购买记录

  • 显示当前权益包的购买记录
  • 包括订单号、购买时间和金额

2.6 底部按钮

  • 转赠好友：将未使用的权益转赠给其他商户
  • 立即使用：跳转至权益使用页面
  • 按钮固定在屏幕底部，确保随时可操作

3. 交互说明

3.1 页面导航

  • 点击返回按钮：返回我的权益页面
  • 点击更多按钮：显示更多操作选项（如投诉、反馈等）

3.2 权益操作

  • 点击"查看记录"：跳转至对应权益的使用记录页面
  • 点击"转赠好友"：显示权益转赠页面
  • 点击"立即使用"：根据权益类型跳转至相应的使用页面

4. 业务逻辑

4.1 权益状态判断

  • 有效期内：当前日期 < 权益到期日期
  • 即将到期：当前日期 < 权益到期日期 <= 当前日期 + 7天
  • 已过期：当前日期 >= 权益到期日期

4.2 使用进度计算

  • 进度百分比 = 已使用次数 / 总次数 × 100%
  • 权益有效期百分比 = 已经过天数 / 总有效天数 × 100%

4.3 转赠规则

  • 只有未使用的权益可以转赠
  • 已经开始使用的权益包不可转赠
  • 转赠后权益立即从当前账户移除

5. 异常处理

5.1 数据加载异常

  • 数据加载过程中显示加载状态动画
  • 网络故障时显示网络错误提示，并提供"重试"按钮
  • 请求超时时（超过15秒）显示超时提示，并提供重试选项

5.2 空数据处理

  • 当权益不存在或已被删除时，显示"权益不存在"提示
  • 提供"返回权益列表"按钮，引导用户回到上一页
  • 在使用进度为0时，展示"未使用"的空状态提示

5.3 操作反馈

  • 权益使用尝试失败时，显示原因并提供解决方案
  • 转赠操作成功/失败时，显示对应的Toast提示
  • 操作中断时（如跳转页面或退出），提示用户是否确认放弃操作

5.4 权限问题

  • 当用户无权使用该权益时，显示权限提示
  • 明确说明获取权限的方式或联系渠道
  • 提供联系客服的快捷入口

5.5 功能限制提示

  • 当权益已过期时，"立即使用"按钮显示禁用状态
  • 当权益已用完时，显示"权益已用完"提示
  • 当转赠条件不满足时，显示具体原因并禁用转赠功能
